# KRecon

KRecon is a Kali Linux compatible reconnaissance tool by Krishna.

## Features
- WHOIS Lookup
- DNS Records Fetch
- Port Scanning
- Subdomain Finder
- HTTP Headers Grabber

## Installation

1. Clone repo  
2. Install dependencies:  
   `pip install -r requirements.txt`  
3. Run tool:  
   `python3 krecon.py --target example.com`
